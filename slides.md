# The Problem

- 70% of Ghanaian farmers rely on middlemen
- Climate change destroys harvests unpredictably
- Language barrier blocks access to modern tools
- No real-time market information
- Lack of expert agricultural advice

---

# Our Solution: AgriAI 2.0

**An intelligent, multilingual, voice-enabled agricultural AI**

- Speaks Twi, Ga, Ewe, Hausa, English & more
- Gives expert farming advice
- Connects farmers to real-time information
- Beautiful white-mode interface

---

# Key Features

- Real Web Search (Tavily)
- Voice Input (OpenAI Whisper)
- Voice Output (ElevenLabs)
- Expert Mode
- Agent Mode
- Crop Disease Detection
- Market Prices & Recommendations

---

# Technology Stack

- **Frontend**: Next.js 16 + TypeScript + Tailwind
- **Backend**: Node.js + Prisma + PostgreSQL
- **AI Models**:
  - Groq (Llama 3.3 70B)
  - OpenAI Whisper
  - ElevenLabs TTS
- **Deployment**: Render.com
